# MyShop Product Page

This project is a product page for an e-commerce website developed using Vue.js. It provides a platform where users can explore detailed product information, view reviews, and make purchases.

## Features
- **Product Gallery:** Supports multiple product images, allowing users to click on thumbnails to view the main product image.
- **Variant Selection:** Users can select different product variants (e.g., color and model).
- **Customer Reviews:** Users can view reviews and ratings from other customers.
- **Add to Cart:** Users can add products to their cart and continue shopping without completing the checkout process.
- **Free Shipping:** Free shipping on all orders.
- **Product Pricing & Discounts:** Product prices and any available discounts are clearly displayed.

## Technologies Used
- **Vue.js:** Frontend framework
- **CSS/SCSS:** For styling and layout adjustments
- **Font Awesome:** For icons
- **Axios:** For API requests
